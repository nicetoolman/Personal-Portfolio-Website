# Personal Portfolio Website

Built with Next.js, Payload CMS, Tailwind CSS, and Cursor-assisted development.
This project powers my official portfolio website:
https://catbox.world

## Tech Stack

- Next.js (App Router)

- Payload CMS

- React

- Tailwind CSS

- Vercel Deployment

- Cursor-assisted coding workflows

## About this project

This portfolio was designed and developed to showcase my creative work, CATBOX series, illustrations, and project documentation.
The website uses a custom modular block system inspired by Payload Website Starter, but heavily modified to fit my own design and content workflow.

## Key Features
- Customizable block-based layout for projects and visual stories

- Responsive design for illustration-focused pages

- CMS-managed publishing workflow

- Sketchlog: a lightweight log system for casual sketches and spontaneous ideas

- Integrated gallery & media components

- Minimal UI focusing on readability and narrative
